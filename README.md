# North Barbershop — București

Pagină de prezentare scroll-driven pentru **North Barbershop**, situat în București, România.

## Previzualizare locală

Deschide `index.html` într-un browser modern, sau servește local:

```bash
python3 -m http.server 8000
# apoi intră pe http://localhost:8000
```
